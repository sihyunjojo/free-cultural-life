# free-cultural-life
도서에 관해서 자유롭게 생각을 나눌 수 있는 웹 커뮤니티


## 핵심 기능
- 도서관련 대화를 나눌 수 있는 커뮤니티(분류별)
- 원하는 도서를 추가하고 그 도서에 대해서 별점과 평가를 3줄이하로 내릴 수 있는 페이지
- 원하는 도서에 대한 톡방(게시판)을 만들고 그 도서에 관해서 이야기할 수 있는 커뮤니티
- 회원 관리 시스템 (전문가, 일반인, 마스터)


## 부가 기능
- 책검색 API
- 베스트셀러 API
- 추천도서 API
- 중고장터 게시판.
